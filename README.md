# Music 4 All

Foundation for a headless music publication platform. The UI is intentionally minimal; it exists to validate the backend and data contracts.

## Requirements
- Node.js 20.x
- pnpm 9.x

## Getting started
```bash
pnpm install
pnpm dev
```

## Scripts
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`

## Health endpoints
- `GET /api/health`
- `GET /api/version`
